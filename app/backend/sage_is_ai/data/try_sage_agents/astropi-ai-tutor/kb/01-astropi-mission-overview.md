## Mission Zero (short)
- Beginner-friendly. Runs on the Astro Pi simulator.
- Output a 30-second message or sensor reading on the LED matrix.
- No special hardware; any computer with a browser works.

## Mission Space Lab (short)
- Team-based research project running on real Astro Pi units in orbit.
- Phase 1: idea + experiment design.
- Phase 2: code an experiment that fits the hardware budget (CPU, memory, time, no internet).
- Phase 3: run on the ISS; download results; write up findings.

## Hardware notes
- Sense HAT exposes temperature, humidity, pressure, gyro, accelerometer, magnetometer, an 8x8 RGB LED matrix, and a small joystick.
- Recent Astro Pi units add a higher-resolution camera and more compute,   but experiments still target the constrained baseline for portability.
